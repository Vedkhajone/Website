Prosthetics Hand

Assistive tech for amputees to regain hand function.

  • EMG/ECG-based gesture recognition.

  • Responsive servo motor control.

  • Adaptive muscle signal mapping.

  • Real-time motion based on biosignals.

  • Easy-to-wear electrode placement.

Web Integration

Control the hand remotely using a smartphone app.

  • ESP32-powered Wi-Fi server.

  • Web-App interface for remote control.

  • Live gesture triggering via browser.

  • Syncs physical hand with UI.

  • JavaScript sliders + control buttons.

3D Printed Design

Designed using CAD and fabricated with affordable PLA material, the hand structure balances cost-efficiency with durability.

  • CAD design using Fusion 360.

  • Lightweight modular finger structure.

  • Integrated servo motor housing

  • Customizable sizing for various users

  • Cost-effective yet structurally strong
